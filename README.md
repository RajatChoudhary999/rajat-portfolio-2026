# Rajat Choudhary | Full Stack Engineer

A high-performance personal portfolio showcasing full-stack engineering projects, technical expertise, and mobile-first design principles. Built to demonstrate clean architecture and scalable logic.

## Core Stack

* **Frontend:** React / Next.js
* **Mobile:** React Native / Expo
* **Styling:** Tailwind CSS / Aesthetic Minimalist UI
* **Deployment:** Vercel / Netlify

## Features

* **Responsive Architecture:** Optimized for desktop, tablet, and mobile displays.
* **Technical Showcases:** Detailed breakdowns of debugging workflows and application builds.
* **Performance Optimized:** High Lighthouse scores with a focus on asset compression and fast load times.
* **Dynamic Routing:** Seamless navigation between project case studies and technical logs.

## Getting Started

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/RajatChoudhary999/rajat-portfolio-2026.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

### Environment Configuration

Create a `.env` file in the root directory for any third-party integrations:
```text
NEXT_PUBLIC_ANALYTICS_ID=your_id_here
```

## Project Structure

* `/components`: Reusable UI elements following atomic design.
* `/hooks`: Custom logic for state management and API interactions.
* `/constants`: Configuration files and static data.
* `/assets`: Optimized images and natural material design elements.
